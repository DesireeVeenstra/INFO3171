Chart Design Guidelines

Our charts should always prioritize clarity and simplicity. Each chart type serves a specific purpose and follows a set of design rules to maintain consistency and avoid unnecessary complexity.

Line Charts

Line charts are used for visualizing trends over time. Ensure the line is smooth, 2px thick, and never curved unless required. Dotted lines can be used to represent secondary metrics such as averages or projections. Dots may be added at key data points to provide extra clarity.

Bar Charts

Bar charts should be used to compare different categories or values. The bars must not have rounded corners and the spacing between bars should always be 50% of the bar width. If using stacked bars, opt for distinct color combinations to enhance readability.

Scatter Plots

Scatter plots are ideal for showing relationships between variables. The points should be 8px by 8px, and if multiple categories are used, vary the shape (circle, square, triangle) to aid differentiation. Ensure that colors from our palette are used to maintain brand consistency.

Meter Charts

Meter charts are used when displaying proportional data where the total sums to 100%. For readability, avoid more than six categories in a single chart. If additional categories are needed, consider grouping smaller ones into an "Other" category.

Grid Lines and Axes

Grid lines should always support the data without drawing too much attention. Use 2px axis lines and grid lines with 50% opacity to create a balance. Keep x-axis labels evenly spaced, and reduce label crowding by adjusting ticks as needed.

Interactive Fly-overs

Fly-overs provide contextual information when users hover or tap on specific data points. These should always be easy to read and strategically placed to avoid obstructing the main data. Make sure fly-overs are responsive on mobile and allow users to interact easily.
